CONNING
- git clone [repo]
- composer install
- .env.example -> .env
- phpmyadmin > new database
- .env > DB_DATABASE=[nama]
- php artisan migrate
- php artisan key:generate
- php artisan serve
good luck :)

TIPS
- nulis ulang kode bisa auto import
- variable kadang hilang
- variable kadang typo
- model fillable
- route jangan pae root ('/') kasih nama ('/[nama]')
